# Random Cloud Animation (HTML, CSS, JavaScript)
A simple and visually pleasing **cloud animation project** built using **HTML, CSS, and JavaScript**.  
The project dynamically generates multiple clouds that move across the screen with **random positions, sizes, speeds, directions, and colors**, creating a natural sky animation effect.


## Features
- **Dynamic Cloud Generation** – Clouds are created using JavaScript.
- **Randomized Properties**
  - Random vertical position
  - Random size (scale)
  - Random speed
  - Random movement direction
  - Random cloud colors
- **Continuous Animation** – Clouds move across the screen repeatedly.
- **Pure Frontend Project** – Built using only HTML, CSS, and JavaScript.
- **Responsive Layout** – Uses viewport units for smooth scaling.


## Technologies Used
- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and animation support  
- **JavaScript (ES6)** – Dynamic cloud creation and randomization  


## How It Works
1. The HTML contains a main container that represents the sky.
2. JavaScript dynamically creates multiple cloud elements.
3. Each cloud receives:
   - A random **position**
   - A random **size**
   - A random **animation speed**
   - A random **direction (left → right or right → left)**
4. CSS styles the clouds and shapes them using rounded elements.
